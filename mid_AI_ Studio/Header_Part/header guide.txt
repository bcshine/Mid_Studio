헤더 만들기

기본가이드>
- 모든 화면은 반응형으로 만들어서, 웹과 모바일 모두에서 잘 보이게 해야 한다. 
- 헤더 안의 내용은 가로로 나란히 정열해아 한다. 
- display: flex로 하라 /* 가로로 나란히 배치해요 */
- align-items: center; /* 세로 가운데로 맞춰요 */
- justify-content: space-between; /* 양쪽 끝과 가운데로 나눠요 */
- gap: 10px; /* 사이사이 간격을 10px로 해요 */
- 폰트는 넥슨폰트 (웹폰트)
- html. css, js 로 만들어 다오. 
- 모든 문장에는 초등생용 주석을 달아다오. 

배치가이드>
- 가장 왼쪽에 Logo (images/logo1.png)
- 그 옆에 브랜드 이름 (중간계AI 스튜디오)
- 그 옆에 차례로, (제품, 브랜드, 고객후기, 기업철학,  인사말) 배치하고 가운데 정열할 것 
- 맨 오른쪽에 로그인 버튼 (흰색바탕, 검은색 테두리, radius 30)
- 가로로 나란히 보기 좋게 정열할 것. 


디자인 기준은 다음과 같다. 
배경색상  : 흰색
정렬 맞추기 :가운데 정렬
박스 사이 간격 : 10
